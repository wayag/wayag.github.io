Person 2 Vector

Posted by Siwei Yang in python   

You have definitively heard about social application, just like Facebook, Instagram, Twitter.

In China the most used socical app is doubtless the WeChat. Maybe it is unnecessary to talk about what In Wechat we can chat to our friend , we can also create a group of friends, and chat to each other within that. In this artical I will analyse the text or words used by individuall talker within a group including my 4 buddies. For privacy protection I will hide all personal data. The central idea is to use topic model like method to describe a person as a vector. All words used by one person will be collected. You can imagine one person as a "document". So it means that in this group there are 5 "documents".

def get_real_cont(content, sep = ':' ):
    ''' content is a list of strings''' 
    
    if str(content) =='nan':

To make the visualization possible , we use 3 as our "topics number". At first we collect all text for each person . So this is the variable corpus_o. And then we transform this into a TF-IDF matrix. Finally we use Non-negative Matrix Factorization to decompose the matrix into two matrice W_o and H_o. Actually W_o is what we need. This is a 5 x 3 matrix. In this way we can express each of my friends as a 3D vector.
As we see, three of them have relatively similar topics whereas the other two have been talking some totally different things.
